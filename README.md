# SignIn-App

**SignIn-App** is a React Native mobile application that replicates an existing interface as part of a Mobile Development course project. The app focuses on a clean sign-in screen with theme toggle functionality and responsive input components.

---

## 🚀 Features

- Modern **Sign-In UI** design
- **Theme toggle** (Light/Dark mode)
- Input for **Name, Email, Phone, Password**
- Modular and reusable components:
  - `InputList`
  - `Button`
  - `ThemeButton`
- Optimized for **Android & iOS**
